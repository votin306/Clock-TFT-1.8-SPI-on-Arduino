# Обычные Часы на Arduino 
### Железо
-Проект собирлся на платформе Arduino Nano(UNO)

-TFT экран SPI 1.8

-Система работает автономно и без управления

### Подключение TFT к Arduino

   LED — 3.3 В - 150 ОМ
   
   SCK — 13 - 1кОМ
   
   SDA — 11- 1кОМ
   
   A0 - 9 - 1кОМ
   
   RESET — 8 - 1кОМ
   
   CS — 10 - 1кОМ
   
   GND — GND
   
   VCC - 5V
### Подключение модуля реального времени
   VCC - 5V
   
   GND - GND
   
   SDA - A4
   
   SCL - A5
